> **Честное название для честного продукта. WinForms + .NET 10 = Боль и страдания.**

Добро пожаловать в **Just Shitty Calculator**. Мы не обещаем вам идеального кода, стабильной работы или логики. Мы обещаем вам C#, калькулятор и несколько функций, которые явно лишние.

Это проект для тех, кто считает, что `2 + 2` должно быть интересным процессом, а не просто результатом. Сборка на самом свежем .NET 10 с использованием классических WinForms гарантирует новые, ранее не виданные баги.

---

## ✨ Основные возможности

Мы добавили функции, о которых вы не просили, и некоторые из них даже работают:

- 🎹 **Встроенное пианино:** Играйте мелодии между вычислениями. Клавиши цифр тоже издают звуки (соседи будут в восторге).
- 🎲 **Генератор кубика:** Встроенный D6. Потому что иногда нужно принять решение важнее, чем расчет налогов.
- 🎨 **Случайный цвет:** Кнопка, которая меняет цвет интерфейса на случайный. Дизайн? Не слышали.
- 🌈 **5 Тем оформления:**
  - 💡 Светлая (Работает нормально)
  - 🌑 Темная **(⚠️ ВНИМАНИЕ: Сломана, текст не читается, глаза болят)**
  - 🔵 Синяя
  - 🟢 Зеленая
  - 🟣 Фиолетовая
- 🌐 **Мультиязычность:** Переключение между 🇷🇺 Русским и 🇬🇧 English. Переводы местами странные, но это добавляет шарма.
- 🎮 **Игра "Угадай число":** Встроена прямо в калькулятор. Прокрастинируйте эффективно!
- 📦 **Переменные:** Можно создавать переменные. Сохранение не гарантировано, но попытка не пытка.

## 🚀 Установка и запуск

### Требования
- Установленный [.NET 10 SDK](https://dotnet.microsoft.com/download) (Удачи в поиске)
- Windows (потому что WinForms, сюрприз!)
- Желание поэкспериментировать
- **Настоятельная рекомендация: Не включайте темную тему**

### Запуск из исходников

```bash
git clone https://github.com/yourusername/just-shitty-calculator.git
cd just-shitty-calculator
dotnet restore
dotnet run
```

## 🐛 Известные проблемы (Known Issues)
Мы верим в прозрачность, поэтому вот список того, что точно сломается:

- 🌑 Темная тема:
  - Текст сливается с фоном.
  - Иногда интерфейс становится полностью черным.
  - WinForms плохо дружит с кастомными темами.
  - Статус: Мы знаем. Мы постараемся. Но нет.
- 🎹 Пианино: Звук может задерживаться или воспроизводиться с разной скоростью.
- 🎲 Кубик: Вероятность выпадения чисел не проверена статистически.
- 🎨 Случайный цвет: Может сгенерировать цвет, от которого болит голова.
- 🔥 .NET 10 + WinForms: Поскольку мы используем самую новую версию фреймворка со старой UI-библиотекой, некоторые баги могут быть вызваны этим сочетанием (мы виним Майкрософт).
- 🪟 Только Windows: Linux и macOS пользователи могут плакать в сторонке.
